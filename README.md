# タスクボード2 メモビューア v0.1.2

タスクボード2のDropbox同期JSONから、メモ欄だけをスマホで閲覧するためのWebアプリです。

## できること

- Dropbox上のタスクボード2同期ファイルを読み込む
- `memoText` だけを表示する
- `- [ ]` / `- [x]` のチェック行を見やすく表示する
- 手動更新
- メモ本文コピー
- PWAとしてホーム画面に追加

## できないこと

- Dropboxへの保存
- メモ編集
- タスク編集
- チェック状態の変更

閲覧専用なので、タスクボード2本体のデータは変更しません。

## 設定項目

### Dropbox App key

Thunderbird版タスクボード2で使っているDropbox App keyと同じものを入れてください。

### 保存ファイルパス

Thunderbird版タスクボード2のDropbox設定にある保存ファイルパスと同じものを入れてください。

旧版の初期値は `/tasks.json` です。  
別のパスにしている場合は、そちらを指定してください。

## 使い方

1. このフォルダ一式をHTTPSで公開します。
   - GitHub Pages
   - Netlify
   - Cloudflare Pages
   - 自分のWebサーバー

2. スマホで公開URLを開きます。

3. `Dropbox設定` を開きます。

4. `Dropbox App key` と `保存ファイルパス` を入力します。

5. `設定保存` を押します。

6. `Dropbox接続URLを開く` を押します。スマホでは同じタブでDropbox認証画面へ移動します。

7. Dropboxで認証し、表示された認証コードをコピーします。

8. ブラウザの戻る操作などで元の画面に戻り、認証コード欄へ貼り付けます。

9. `認証コードを保存` を押します。

10. メモ欄が表示されます。

11. 必要に応じてスマホの「ホーム画面に追加」を行います。

## 注意

- Dropboxのrefresh tokenは、このWebアプリを開いたブラウザのlocalStorageに保存されます。
- 共有端末では使わないでください。
- URLを公開しても、Dropbox認証を通らない限りメモは読めません。
- ただし、Webアプリ自体を置く場所は自分で管理できる場所を推奨します。


## v0.1.1 変更点

## v0.1.2 変更点

- `memoText` 以外の保存形式も探索するようにしました。
- 読み込んだJSONのルートキー、memo系候補、実際のメモ取得元を確認できる「読込診断」を追加しました。
- Dropbox接続は成功しているがメモが表示されない場合の切り分けに使えます。
